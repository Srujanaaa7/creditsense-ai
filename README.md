# CreditSense AI

CreditSense AI is a machine learning-based web application that predicts the credit risk of a loan applicant.

## Features
- Predicts loan risk (Low / High)
- Real-time analysis
- User-friendly interface

## Technologies Used
- Python
- Flask
- Machine Learning (Logistic Regression)
- NumPy, Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## Dataset
- credit_risk_dataset.csv
